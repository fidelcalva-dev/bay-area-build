# Self-Selling Quote Format

## Philosophy

The quote is not a calculator — it's a guided proposal that:
1. Recommends the right solution
2. Shows transparent pricing
3. Builds trust at every step
4. Captures rich intent for Sales follow-up
5. Acts as a sendable proposal document

## Customer-Facing Proposal Sections

### 1. Service Details
- Dumpster size with badge (Recommended / Budget / More Capacity)
- Project type
- Material type (customer-friendly label)
- Rental period

### 2. Additional Services (if any)
- Swap / Dump & Return
- Same-Day Delivery
- Special Placement
- Required Disposal Site

### 3. Location
- ZIP / Address
- Nearest yard + distance
- Placement type

### 4. Pricing Breakdown
- Base price range
- Delivery & Pickup: Included
- Disposal: Included
- Weight allowance: Included
- Overage rate (if applicable)
- **Total Estimate**

### 5. Key Policies
- Rental term + extra day fee
- Overage rate
- Fill line rules
- No hazardous materials

### 6. Contact Information
- Customer name, phone, email
- Company (if provided)
- Customer notes (if provided)

### 7. Next Steps
- Confirm Order button
- Terms acceptance
- Privacy / Terms links

## Trust Elements Throughout
- "Licensed & Insured"
- "Real Local Yard"
- "Transparent Pricing"
- "15-min response"
- "Bay Area since 2009"
- "No hidden fees"

## Option A/B/C Format (Phase C)

When multiple options exist:
- **Option A** = Lowest Cost (smaller size)
- **Option B** = Recommended (best fit)
- **Option C** = More Capacity (larger/safer)

Each option shows: size, price range, best-for guidance, included tonnage.
