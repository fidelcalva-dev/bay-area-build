## Assessment Criteria

A page is only created if it meets ALL of:
1. Inside Bay Area scope (9 counties)
2. Useful keyword target with search volume
3. Distinct intent (not duplicate of existing page)
4. Strong conversion role
5. Source data is accurate and current
